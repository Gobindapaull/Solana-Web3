# ▶️ How to Run the Bot

### 1. Extract the ZIP

* Right-click the ZIP file
* Click **Extract Here** or **Extract All**

---

### 2. Open Folder in Terminal

* Open the extracted folder
* In the address bar, type `cmd` and press Enter
  *(or open terminal manually in that folder)*

---

### 3. Install Dependencies (if needed)

```bash
npm install
```

---

### 4. Configure the Bot

* Add your:

  * Private Key
  * RPC URL
  * Destination Wallet

---

### 5. Start the Bot

```bash
node bot.js
```

---

### ✅ Done

* The bot will start running automatically
* Keep the terminal open while it's running

---

### ⚠️ Important

* Do NOT close the terminal
* Use a VPS for 24/7 running
* Test with a small amount first
